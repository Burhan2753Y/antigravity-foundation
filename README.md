# antigravity-foundation
A production-ready starter template for Antigravity IDE. Pre-configured with Model Context Protocol (MCP) definitions, agentic workflow rules, and optimized VS Code settings for AI-assisted engineering.

# Antigravity Foundation

To start a new project using this blueprint without linking to this repo:

1. mkdir my-new-app && cd my-new-app
2. git clone --depth=1 https://github.com/burhan2753y/antigravity-foundation.git .
3. rm -rf .git
4. git init
